# Healthcare Machine Learning Care Gap Portfolio

This portfolio demonstrates machine learning projects focused on healthcare quality analytics, HEDIS care gaps, patient outreach prioritization, and value-based care improvement.

## Projects

### 1. HEDIS Care Gap Outreach Prioritization

Built a machine learning model to predict whether a patient is likely to have an unresolved care gap and should be prioritized for outreach.

Skills demonstrated:

- Synthetic healthcare data generation
- Feature engineering
- Classification modeling
- Missing value imputation
- Model evaluation
- Healthcare quality analytics
- HEDIS care gap logic
- Outreach prioritization
